## 1. 機能概要

### 1.1 目的

経営会議（月次経営会議、取締役会等）の準備・実施・記録を支援する**会議運営プラットフォーム**。

### 1.2 ターゲット会議体

| 会議体 | 頻度 | 特徴 |
|--------|------|------|
| 月次経営会議 | 月1回 | メインターゲット |
| 取締役会 | 月/四半期 | 非上場は経営会議と統合も |
| 事業部内会議 | 隔週など | 見込管理、部門内レビュー |

### 1.3 主要機能

| 機能カテゴリ | 内容 |
|--------------|------|
| 会議設定 | 会議種別、対象期間、締切日、報告対象部門の設定 |
| 部門報告 | 動的フォームによる部門からの報告登録、見込コメント引用 |
| レポート閲覧 | **固定タブ構成** (サマリー、部門報告、KPI・AP、詳細分析、前回比較、議事録) + **カスタマイズ可能ダッシュボード** (コンポーネント配置) |
| KPI/AP連携 | KPIカード表示、アクションプラン進捗、カンバン/ガント遷移 |
| 議事録 | 会議終了後の決定事項・議事録記録、添付ファイル |
| スナップショット | 会議時点の数値・KPI・AP進捗をJSON保存、前回比較 |

---

## 2. 会議運営フロー

```
1. 会議設定（企画部門）
   └─ 会議種別・対象期間・締切日を設定

2. 部門報告登録（各部門）
   └─ 締切までに報告を登録
   └─ 見込入力のコメントを引用可能

3. 締切・確認（企画部門）
   └─ 締切後に内容確認・取りまとめ

4. 事前展開（役員・監査役）
   └─ 会議前に資料を展開・事前確認

5. 会議実施
   └─ ダッシュボード表示
   └─ 数値＋コメント＋KPI状況を確認

6. 会議後記録
   └─ 議事録・決定事項の記録
   └─ スナップショット保存（バッチで自動取得）
```

## 3. 比較軸

| 優先度 | 比較パターン | 用途 |
|--------|--------------|------|
| ① | 予算 vs 実績 vs 見込 | 予算達成度＋着地見通し |
| ② | 前月見込 vs 今月見込 | 見通しの変化（1ヶ月推移） |
| ③ | 先月登録見込 vs 当月実績 | 見込精度確認 |
| ④ | 前月/前年/前期比較 | オプション |

## 7. 標準テンプレート：月次経営会議

### 7.1 報告フォーム構成

| セクション | 項目 | タイプ | 必須 |
|------------|------|--------|------|
| 業績サマリー | 売上見通し | SELECT（好調/計画通り/懸念/要対策） | ○ |
| | 利益見通し | SELECT（好調/計画通り/懸念/要対策） | ○ |
| | サマリーコメント | TEXTAREA（200字程度） | ○ |
| 差異要因 | 売上差異の主要因 | TEXTAREA（引用可） | ○ |
| | 粗利差異の主要因 | TEXTAREA（引用可） | ○ |
| | 販管費差異の主要因 | TEXTAREA（引用可） | - |
| リスク・課題 | 主要リスク | TEXTAREA | - |
| | 対応状況 | TEXTAREA | - |
| アクション | 今月の重点施策 | TEXTAREA | ○ |
| | 経営への要請事項 | TEXTAREA | - |
| 添付資料 | 補足資料 | FILE（複数可） | - |

### 7.2 設計意図

- **入力負荷軽減**: 10分以内で入力完了できる項目数
- **見通し把握**: SELECT で直感的に状況を把握
- **因果追跡**: 売上→粗利→販管費の段階利益構造で差異を整理
- **柔軟性**: 任意項目で詳細を補足可能

## 8. 技術仕様

### 8.1 スナップショット

- **保存形式**: JSON（DBコンパクト化のため）
- **取得タイミング**: 会議日設定後にバッチで自動取得
- **種別**: FACT_SUMMARY / KPI_STATUS / ACTION_PROGRESS

### 8.2 引用機能（パターンB）

- TEXTAREA に引用テキストを挿入
- `quote_refs_json` で引用元メタデータを保持
- 引用元: 下位部門報告 / 見込バージョンコメント

### 8.3 会議ステータス遷移

```
DRAFT → OPEN → COLLECTING → DISTRIBUTED → HELD → CLOSED → ARCHIVED
```
