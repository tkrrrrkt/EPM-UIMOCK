# EPM/FP&A レポーティング・ダッシュボード仕様書

## エグゼクティブサマリー

本レポートは、日本の中堅企業（売上高100億〜1,000億円）向けEPM SaaSプロダクトにおける「真に使える」レポーティング・ダッシュボード・分析機能の定義を目的とした包括的調査結果である。**2024-2025年の最新情報**に基づき、国内外12社以上のEPMベンダーを比較分析し、ペルソナ別レポートカタログ、ダッシュボードテンプレート、ベンチマークマトリクス、要件定義、ロードマップを策定した。

主要な発見として、**AI/ML機能の急速な進化**（特にアジェントAI、異常検知、ナラティブ生成）、**Excel中心文化への対応**の重要性、そして**ナラティブレポーティング**が経営報告の差別化要因となっていることが挙げられる。

## (2) ダッシュボードテンプレートパターン（8種）

### テンプレート1: 経営者コックピット（Executive Cockpit）

**対象ペルソナ**: CEO/COO/CFO **用途**: 毎日/毎週の経営状況把握

**レイアウト構成**:

```
┌─────────────────────────────────────────────────────────┐
│  [ヘッダー] 期間セレクター | 事業セレクター | 更新日時  │
├─────────────────┬─────────────────┬─────────────────────┤
│   売上高KPIタイル  │  営業利益KPIタイル │   現金残高KPIタイル   │
│  ▲12% vs 予算    │   ▼3% vs 予算    │   ¥5.2B          │
├─────────────────┴─────────────────┴─────────────────────┤
│              売上高推移グラフ（実績 vs 予算 vs 前年）            │
├─────────────────────────────┬───────────────────────────┤
│     事業別営業利益貢献（横棒）    │   トップ5 アラート一覧      │
├─────────────────────────────┴───────────────────────────┤
│              P/Lブリッジ（予算→実績ウォーターフォール）           │
└─────────────────────────────────────────────────────────┘
```

**主要ウィジェット**:

- KPIスコアカードタイル（売上高、営業利益、EBITDA、現金残高）
- トレンドチャート（折れ線：実績/予算/前年）
- ウォーターフォールチャート（予実ブリッジ）
- アラートリスト（閾値超過項目）

**インタラクション**:

- グローバルフィルター（期間、事業、地域）
- KPIタイルクリック→詳細ダッシュボードへドリルスルー
- チャートポイントクリック→取引明細へドリルダウン

**権限設計**: 経営層のみ全社データ閲覧可能、事業部長は担当事業のみ

**共有方法**: PDF自動生成（毎朝7時）、メール配信、モバイルプッシュ通知

## (4) 要件定義ドラフト

### 機能要件（ユーザーストーリー + 受入基準）

#### A. ダッシュボード・可視化

|ID|ユーザーストーリー|受入基準|優先度|
|---|---|---|---|
|FR-D01|CEOとして、全社業績を一目で把握できるエグゼクティブダッシュボードが欲しい|KPIタイル5種以上表示、3秒以内描画、モバイル対応|MVP必須|
|FR-D02|CFOとして、P/L予実差異をウォーターフォールで視覚化したい|予算/実績/着地見込の3バージョン比較、正負の色分け、ドリルダウン|MVP必須|
|FR-D03|事業部長として、担当事業のKPIツリーを見て改善レバーを特定したい|3階層以上のドライバー分解、What-if連動、シナリオ比較|Differentiator|
|FR-D04|経営企画として、ローリングフォーキャストの進捗を一覧で把握したい|部門別進捗率、期限アラート、バージョン比較|Differentiator|
|FR-D05|トレジャリーとして、運転資本（CCC）をリアルタイムで監視したい|DSO/DPO/DIO/CCC自動計算、トレンド表示、取引先ドリルダウン|Differentiator|
|FR-D06|経営企画として、ROIC分解ツリーで資本効率を可視化したい|ROIC→NOPAT×回転率、WACC比較、事業別比較|Future|

#### B. レポート・ナラティブ

|ID|ユーザーストーリー|受入基準|優先度|
|---|---|---|---|
|FR-R01|CFOとして、予実分析レポートをExcelで作成・更新したい|Excel Add-in連携、ワンクリック更新、書式保持|MVP必須|
|FR-R02|経営企画として、取締役会資料を効率的に生成したい|レポートブック機能、PDF/PPT出力、ナラティブ統合|Differentiator|
|FR-R03|部門長として、差異に対するコメントを入力・承認したい|セル/行レベルコメント、@メンション、承認ワークフロー|MVP必須|
|FR-R04|CFOとして、AIが差異要因を自動で説明してほしい|GenAI連携、Top3要因抽出、日本語対応|Differentiator|
|FR-R05|経理として、レポートをスケジュール配信したい|日次/週次/月次スケジューラー、PDF/Excelバースティング|MVP必須|
